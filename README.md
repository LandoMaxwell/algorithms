# Algorithms (Python)

Python 算法实现集合，包含面试常见算法题解和工具脚本。

## 目录

| 文件 | 描述 |
|------|------|
| `binary_tree_inorder_traversal.py` | 二叉树中序遍历 |
| `first_missing_positive.py` | 缺失的第一个正数 |
| `longest_palindromic_substring.py` | 最长回文子串 |
| `merge_intervals.py` | 合并区间 |
| `product_of_array_except_self.py` | 除自身以外数组的乘积 |
| `rotate_array.py` | 旋转数组 |
| `set_matrix_zeroes.py` | 矩阵置零 |
| `skew_heap_reconstruction.py` | 斜堆重构 |
| `folder_renamer.py` | 批量文件重命名工具 |
| `test_skew_heap.py` | 斜堆测试 |

## 算法分类

### 数据结构
- **二叉树**: 二叉树中序遍历
- **堆**: 斜堆重构

### 字符串
- 最长回文子串

### 数组
- 缺失的第一个正数
- 合并区间
- 除自身以外数组的乘积
- 旋转数组
- 矩阵置零

### 工具
- 批量文件重命名工具

## 运行测试

```bash
# 运行斜堆测试
python test_skew_heap.py

# 运行文件重命名工具
python folder_renamer.py
```
